README

Included Files:
--------------------------------------------------------------------------------
py2py Source
+Downloads				Folder for storing files downloaded by user
+Generated				Folder for files created by PyQT5
+Resources				Folder for holding GUI images
+UI						Folder for storing PyQT5 Creator files
make.bat				Run this script to rebuild GUI files
-connectiondialogue.py	File that prompts the user for a peer at the beginning
-main.py				Use this to start the program		
-mainwindow.py			This is the class that handles the user's main GUI
-peer.py				This is a generic peer class
-peerconnection.py		This class handles the connection between two peers
-peerfilemanager.py		This inherits the peer class and adds some file 
						management features
--------------------------------------------------------------------------------

Programs needed to run: 
--------------------------------------------------------------------------------
Python 2.7
PyQT5.4.1
Note that, depending on your operating system, you may need to install 
additional programs so that these programs function.
--------------------------------------------------------------------------------

To compile and run this source code:
--------------------------------------------------------------------------------
1. Run make.bat
2. Run main.py
--------------------------------------------------------------------------------
